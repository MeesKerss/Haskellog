
\section{Conclusion}\label{sec:Conclusion}
Haskellog successfully imitates Prologs basic functionalities. Our QuickCheck tests show that 
the program at least doesn't crash when fed strings representing rules our quiries. With more time we would maybe add the possibility to negate 
formulas.\\
We originally planed to test our implementation with the an already 
existing Haskell implementation of Prolog\cite{haskellprolog}. The problem was that this implementation runs on an older version of Haskell. 
Although we didn't find any bugs by simply coming up 
with queries and set of rules, we would be more secure 
in our product if we did some randomized testing of the 
statement it returns.\\
This project was great conclusion to the beginning of journey with functional programming. We were able to make
use of the state monad and pattern matching, which were novel to us. We are looking forward to keep dabbling in functional programming in the future.
